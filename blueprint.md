# Lotto Number Generator

## Overview

A web application that generates random lottery numbers for the user. The application will be designed with a modern and visually appealing interface, ensuring a great user experience.

## Style, Design, and Features

### Visual Design
*   **Aesthetics:** A unique user experience that incorporates modern components, a visually balanced layout with clean spacing, and polished styles.
*   **Responsiveness:** The app will be mobile-responsive, adapting to different screen sizes for both mobile and web.
*   **Color Palette:** A vibrant and energetic color palette will be used.
*   **Typography:** Expressive and relevant typography will be chosen to emphasize key information.
*   **Iconography:** Icons will be used to enhance user understanding and navigation.
*   **Interactivity:** Interactive elements like buttons will have a "glow" effect and shadows to create a sense of depth.

### Features
*   **Number Generation:** Generate 6 unique random numbers between 1 and 45.
*   **Display:** Display the generated numbers in a clear and visually appealing format.
*   **User Interface:**
    *   A "Generate Numbers" button to trigger the number generation.
    *   A dedicated area to display the results.

## Current Implementation Plan

1.  **`index.html`:** Create the basic structure for the website, including a header, a button to generate numbers, and a container to display the numbers.
2.  **`style.css`:** Style the website with a modern design, including a custom font, a visually appealing color scheme, and responsive layout.
3.  **`main.js`:** Implement the JavaScript logic to generate and display the lottery numbers when the button is clicked.
